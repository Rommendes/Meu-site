Poetizar a web

POETIZAR A WEB É UM PRAZER,
UM DEVER DO WEB POETA
SE ELE ASSIM NÃO FIZER,
A VIDA FICA SEM COR
ELEVA MUROS INDECIFRÁVEIS
TRAVA SUAS IDÉIAS
NÃO CODIFICA SEUS CÓDIGOS

POETIZAR A WEB É CONSTRUIR VÍNCULOS
ATRAVÉS DOS CÓDIGOS
DE MANEIRA VIVA E SUBJETIVA,
É TRAZER O ALEGRIA COM ALGORITMOS

POETIZAR A WEB É DERRUBAR MUROS
TRAZER SENTIDO
DE FORMA QUE A VIDA
SEJA CODIFICADA COM LEVEZA.
